# Fire Chicks

A chaotic asymmetric multiplayer game that transforms the childhood game 麻鷹捉雞仔 (Eagle Catches Chicks) into a digital experience set in HKUST's academic culture — where the Eagle hunts, the Chicks cooperate, and the system decides who matters.

**Play at:** [firechicks.vercel.app](https://firechicks.vercel.app)  
**Exhibition Page:** [EMIA 2200 — Fire Chicks](https://emia2200.hkust.edu.hk/student-works-fire-chicks)

---

## How It Works

- **1 Host** (big screen) creates a room and controls the game flow
- **3+ Players** (phones) join via room code and take roles as Fire Chicks or the Eagle
- Chicks cooperate to collect exam tips, share knowledge, and survive
- The Eagle hunts them down — but only after Stage 1
- Mini-games (Mock Exams, Hitbox Challenges, Crossy Road) disrupt the chaos
- It all ends with a Final Exam... and a Transcript that reveals your true value
